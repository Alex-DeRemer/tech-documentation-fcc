A element

In short, this is a tag that helps you make a link to a web page, to your page, social media site, product within online store, etc... The main attribute for this HTML tag is the href (Hypertext Reference) attribute, where you put a link to a website you want to link to. The other attribute is target, it can be used to open a link in a new window, so that users do not lose focus on the current page.

An example of an a element:

Click here to <a href="https://google.com">search</a> anything you want!

Span element

The span tag groups text for which we want to have different styling. A good example would be if we wanted to have a red word inside a sentence.

An example of an span element:

I want to make this word <span>RED!</span>

The CSS required to style the span element:

span {
color: red;
}

Img element

We use the IMG tag to show images. Considering that images can be within files for your web page or anywhere online. In addition, the most important attribute is src (source), which tells where the picture is. It is important to know that img elements are self-closing elements. (This means they only need one tag that closes itself.)

An example of an img element:
(This code will show this cute cat photo!)

<img href="https://img.freepik.com/free-vector/cute-happy-cat-laying-cartoon-vector-icon-illustration-animal-nature-icon-concept-isolated-premium-vector-flat-cartoon-style_138676-3738.jpg?t=st=1654520785~exp=1654521385~hmac=a53f3eee1b1470541e20468cf9fdce6850b6a736938130d36d4d21567eae5188&w=740" alt="cute cat"/>

Div element

An element which is mostly used to group elements and to act as a template for new controls. The div HTML tag is an element you will use to divide a significant part of an HTML document from other parts. For example, imagine you have a list of products on your web page, you would use a div for each product.

Ol element

Ordered list. Also includes <LI> list items, which will be numbered automatically. As a result, this is very useful if you add more list items, because the numbers will always be in the right order.

An example of an ol element:

<ol>
<li>1.Item One</li>
<li>2.Item Two</li>
<li>3.Item Three</li>
<li>4.Item Four</li>
<li>5.Item Five</li>
</ol>

The result of this code will look something like this...

  1. Item One
  2. Item Two
  3. Item Three
  4. Item Four
  5. Item Five
